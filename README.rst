Installation
================================================================================

You can clone it and install it:

.. code-block:: bash

    $ git clone http://github.com/chfw/Minion.git
    $ cd Minion
    $ python setup.py install


Evaluation
================================================================================

After installation, you can evalute it from current Minion directory:

.. code-block:: bash

    $ minion public
